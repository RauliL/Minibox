## Compile

The program must be compiled with GCC.

### MS-DOS, FreeDOS, etc.

You must use DJGPP.

You should be compiled this command line is given below.

    make dos

## Usage

    MINIBOX Command [Arguments]
    MINIBOX --help|--version

## Commands

* beep
* cat
* cd
* clear
* cp
* date
* echo
* help
* ls
* mkdir
* mv
* pwd
* rm
* rmdir
* sh
* time
